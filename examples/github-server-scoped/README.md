# MCP Server for github-scoped

MCP server for github-scoped (with 1099 capabilities)

Generated by [mcp-anything](https://github.com/gabrielekarra/mcp-anything).

## Installation

```bash
pip install -e .
```

## Configuration

### Upstream service connection

This server proxies requests to an upstream HTTP service. Set this before starting:

```bash
export GITHUB_SCOPED_BASE_URL=http://localhost:0
```

**The upstream service must already be running** before you start this MCP server.

## Usage

### Start the server

```bash
mcp-github-scoped
```

### Add to your MCP client

```json
{
  "mcpServers": {
    "github-scoped": {
      "command": "mcp-github-scoped",
      "args": []
    }
  }
}
```

The generated `mcp.json` also includes the `env` block for the upstream base URL and auth placeholders.
