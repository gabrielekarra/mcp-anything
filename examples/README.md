## ffmpeg

**Source:** [ffmpeg](https://ffmpeg.org/) â€” the universal video/audio processing tool. No REST API, no OpenAPI spec â€” just a massive CLI with 197 flags.

**How it was generated:** Captured `ffmpeg --help full` output to a `.txt` file. mcp-anything detected the help file, parsed 101 option groups (codecs, muxers, demuxers, filters, devices), and generated one tool per group with typed parameters.

**Command:**
```bash
ffmpeg --help full > /tmp/ffmpeg-src/help.txt
mcp-anything generate /tmp/ffmpeg-src --name ffmpeg --no-llm
```

**Generated tools (101):** Grouped by ffmpeg section â€” `ffmpeg_global_options`, `ffmpeg_video_options`, `ffmpeg_audio_options`, `ffmpeg_subtitle_options`, codec-specific options (AAC, FLAC, HEVC, Opus, ProRes), muxer options (HLS, MPEGTS, WebM, MKV), demuxer options, device I/O options, and more.

**mcp.json config:**
```json
{
  "mcpServers": {
    "ffmpeg": { "command": "mcp-ffmpeg", "args": [] }
  }
}
```

**Output:** [`ffmpeg-server/`](ffmpeg-server/)

## Generating your own

```bash
pip install mcp-anything
mcp-anything generate /path/to/any/project --name my-project
```

The generated server will be in `mcp-my-project-server/`. Install it with `pip install -e mcp-my-project-server/` and add the mcp.json config to your MCP client.
