### httpstat

**Source:** [reorx/httpstat](https://github.com/reorx/httpstat) ‚Äî Python CLI that visualizes curl timing statistics.

**Command:**
```bash
mcp-anything generate /path/to/httpstat --name httpstat --no-llm
```

**What was detected:** CLI application with argparse (2 source files)

**Generated tools (2):**
| Tool | Strategy | Works? |
|------|----------|--------|
| `run_httpstat` | `cli_subcommand` ‚Äî executes `python httpstat.py <args>` | Yes |
| `print_help` | `cli_subcommand` ‚Äî prints help output | Yes |

**Verified:**
```

‚Üí DNS Lookup: 63ms | TCP Connection: 197ms | TLS Handshake: 431ms | ...
```

**Output:** [`httpstat-server/`](httpstat-server/)

## Generating your own

```bash
pip install mcp-anything
mcp-anything generate /path/to/any/project --name my-project
```

**Works with:** CLI tools (argparse, Click, Typer), REST APIs (FastAPI, Flask, Spring Boot), OpenAPI specs, Python libraries with classes. All produce fully working servers out of the box.
