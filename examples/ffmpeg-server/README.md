# MCP Server for ffmpeg

MCP server for ffmpeg (with 101 capabilities)

Generated by [mcp-anything](https://github.com/gabrielekarra/mcp-anything).

## Installation

```bash
pip install -e .
```

## Usage

### As an MCP server (stdio transport)

```bash
mcp-ffmpeg
```

### Claude Desktop configuration

Add to your `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "ffmpeg": {
      "command": "mcp-ffmpeg"
    }
  }
}
```
