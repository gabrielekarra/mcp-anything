# MCP Server for imagemagick

MCP server for imagemagick (with 200 capabilities)

Generated by [mcp-anything](https://github.com/gabrielekarra/mcp-anything).

## Installation

```bash
pip install -e .
```

## Usage

### As an MCP server (stdio transport)

```bash
mcp-imagemagick
```

### Claude Desktop configuration

Add to your `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "imagemagick": {
      "command": "mcp-imagemagick"
    }
  }
}
```
